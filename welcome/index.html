localLEPTON_114

Diese lokale LEPTON 1.1.4 Installation verwendet den USBWebserver 8.6 mit PHP 5.4.17 sowie MySQL 5.6.13.

Mit phpinfo() erhalten Sie ausführliche Informationen über die aktuelle PHP Konfiguration.

Sie haben Zugriff auf das LEPTON Backend mit:

admin
carlotta

Sie erreichen den phpMyAdmin über http://localhost:8080/phpmyadmin, die Zugangsdaten lauten:

root
usbw

Bei Fragen und Problemen erhalten Sie eine Unterstützung über die Support Group der phpManufaktur.

Sie können diese Information jederzeit mit http://localhost:8080/welcome aufrufen.
